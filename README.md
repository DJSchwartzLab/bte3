# Survey of _Bacteroides fragilis bte2_ & _bte3_ in human stool metagenomes
This repository contains bash scripts and other files needed to reproduce the survey of _B. fragilis_ T6SS effectors _bte2_ & _bte3_ across multiple large-scale human stool microbiome datasets, as described in Valguarnera _et al_. (under review).
